About This Project

Football Player Statistics & Predictions Dashboard

A comprehensive Machine Learning project that predicts football player performance and visualizes statistics through an interactive web dashboard.

Features

ML Model

Random Forest Regressor to predict player performance ratings

Analytics

Comprehensive statistical analysis and trend visualization

Heatmaps

Field coverage visualization showing player positioning

Radar Charts

Multi-dimensional performance analysis

Predictions

Real-time performance predictions with interactive controls

Match Data

Detailed match-by-match statistics tracking

Technologies Used

Backend

  • Python 3.8+
  • Flask Web Framework
  • Scikit-learn (ML)
  • Pandas & NumPy

Frontend

  • HTML5 & CSS3
  • JavaScript (ES6+)
  • Chart.js
  • Plotly.js

Machine Learning

  • Random Forest Regressor
  • Feature Engineering
  • Cross-validation
  • Model Evaluation

Model Details

Algorithm: Random Forest Regressor with 100 estimators

Features Used (10):

  • Minutes played
  • Goals scored
  • Assists
  • Total shots
  • Shots on target
  • Passes completed
  • Pass accuracy (%)
  • Tackles made
  • Interceptions
  • Dribbles completed

Target: Performance rating (0-100 scale)

Performance: R² Score > 0.95 on test set with low RMSE

Data

This project uses synthetic data that mimics realistic football statistics:

  • 50+ players across 4 positions (Forward, Midfielder, Defender, Goalkeeper)
  • 20 matches per player (1000+ total records)
  • Position-specific statistical distributions
  • Realistic performance ratings based on multiple factors

Use Cases

Player Analysis

Comprehensive evaluation of player performance across multiple metrics

Scouting

Predict player potential based on statistical performance

Team Strategy

Understand player strengths and weaknesses for tactical decisions

Educational

Learn ML and data visualization techniques in sports analytics

To run this project locally:
